[![GitHub](https://img.shields.io/badge/GitHub-stouffer--labs%2Fretrivio-blue?logo=github)](https://github.com/stouffer-labs/retrivio)

## Overview

Retrivio builds an AI-ready understanding of your local codebase using AST-aware chunking, hybrid search, and graph-based context expansion. It is designed for fast terminal workflows as well as MCP and API integrations.

## Quick Start

```bash
retrivio install
retrivio add ~/c-projects
retrivio index
retrivio pick --query "storage replication"
```
